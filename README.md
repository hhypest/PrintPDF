# PrintPDF

## Программа для пакетной печати pdf-файлов.

1. Выберите принтер
2. Выберите каталог
3. Выберите файлы в каталоге
4. Печатайте (необходим принтер, поддерживающий Raw печать)

## Не поддерживается:

1. Настройка печати
